# نظام إدارة تأجير العقارات — خطة المهام الكاملة

## ⚠️ قرارات ثابتة — لا تتغير تحت أي ظرف

```
╔══════════════════════════════════════════════════════════════╗
║  هذه القرارات نهائية ومغلقة — لا نقاش فيها                  ║
║                                                              ║
║  Frontend  →  React + TypeScript                             ║
║  UI        →  Ant Design (antd) — RTL                        ║
║  Backend   →  Supabase (لا يوجد backend مخصص)               ║
║  Database  →  PostgreSQL عبر Supabase                        ║
║  Hosting   →  Vercel (frontend) + Supabase (DB)              ║
║                                                              ║
║  ❌ لا تقترح تغيير أي من هذه القرارات                        ║
║  ❌ لا تستبدل Supabase بـ Firebase أو غيره                   ║
║  ❌ لا تستبدل React بـ Vue أو Next.js أو غيره                ║
║  ❌ لا تستبدل PostgreSQL بـ MongoDB أو غيره                  ║
╚══════════════════════════════════════════════════════════════╝
```

---

## قواعد التطوير الثابتة

- **لا localhost** — لا تشغّل أي dev server أثناء البناء
- **لا تعديل على Stack** — القرارات التقنية مغلقة
- **بعد كل مهمة** — راجع الـ checklist قبل الانتقال للتالية
- **لغة الواجهة** — عربي أساسي، `dir="rtl"` دائماً
- **بناء فقط** — `npm run build` للتحقق، لا `npm run dev`
